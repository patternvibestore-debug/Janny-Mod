# Minecraft Mod Installation Guide

This repository provides a simple guide for installing Minecraft mods safely on Windows.

## What You'll Learn

* How to install Minecraft Forge
* Where to place mod files
* Common installation errors
* Version compatibility tips
* Safe downloading practices

## Installation Steps

1. Install Java.
2. Download the correct version of Minecraft Forge.
3. Place the mod's `.jar` file inside the `mods` folder.
4. Launch Minecraft using the Forge profile.
5. Verify that the mod loads correctly.

## Common Issues

* Incorrect Minecraft version
* Wrong Forge version
* Missing dependencies
* Corrupted mod files
